LA4090V101 REVISION NOTES:
Version 1.01 replaces Version 1.  Substantial inconsistencies in the variable names in Version 1 
were eliminated by nesting the subtotal variables within the total variables, especially for the 
occupational variables.  The "AREA2" variable was deleted after determining that its units are 
unknown.  Version 1.01 has the detailed data description sheets, whereas Version 1 does not.  
Users should verify that they have version 1.01 by looking at the main variable table in the USER'S GUIDE.   
It should have two columns listing the variables names, with subtotal variables identified.
